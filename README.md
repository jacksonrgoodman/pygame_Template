# Base Python Project
### all set to start from scratch 8-)
---

#### ___its dangerous to go alone, take this:___

1. Enter source directory:
    ```
    cd src/
    ```
2. Setup Environment:
    ```
    python -m venv .env
    ```
3. Activate:
    ```
    .env\scripts\activate
    ```
4. After Install:
    ```
    pip freeze > requirements.txt
    ```
5. On Another Machine:
    ```
    pip install -r requirements.txt
    ```
    
### Check Yourself:
    pip list
